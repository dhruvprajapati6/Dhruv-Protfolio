## 🚀 Quick Start

<br/>

```bash
# 📥 Clone this repo
git clone https://github.com/dhruvprajapati6/portfolio.git

# 📂 Enter the project
cd portfolio

# 📦 Install all dependencies
npm install

# 🔥 Start dev server
npm run dev

# 🌐 Open in browser
# → http://localhost:5173
```

<br/>

> 💡 **Requirements:** Node.js 18+ · npm 9+

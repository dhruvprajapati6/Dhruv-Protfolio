# 🚀 INSTALLATION GUIDE

## 📥 Clone Repository

```bash
git clone https://github.com/your-username/portfolio.git
```

## 📂 Open Folder

```bash
cd portfolio
```

## 📦 Install Packages

```bash
npm install
```

## ▶️ Start Development Server

```bash
npm run dev
```

## 🌍 Build Production Version

```bash
npm run build
```
